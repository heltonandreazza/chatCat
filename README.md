# chatCat
A chat web app build with NodeJs with focus in learning and pratice SocketIO and Nodejs.

The app uses:

- EJS
- Nodejs
- ExpressJs
- SocketIO
- Redis
- CommonJS
- Heroku
- Facebook API
- Twitter API

Demo available:
https://chat-cat-app-helton.herokuapp.com/
